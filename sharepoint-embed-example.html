To embed this app in SharePoint:

  1. Replace YOUR_DEPLOYED_URL_HERE below with your actual deployed app URL
  2. In SharePoint, edit a page and add an Embed web part
  3. Copy the iframe code below and paste it into the Embed web part
  4. Adjust the height value (currently 1200px) based on your needs
  5. Save and publish your SharePoint page

Code to copy:

<iframe src="YOUR_DEPLOYED_URL_HERE" width="100%" height="1200px" frameborder="0" allow="camera; clipboard-write"></iframe>
